# Ceremonies

## Configured Ceremonies

### Pre-Ship Review
- **trigger:** manual
- **when:** before merging to main / publishing a package
- **facilitator:** Holden
- **participants:** Naomi, Amos, Bobbie, Rai
- **purpose:** Confirm tests pass, build is clean, NuGet metadata is correct, no RAI issues

### Weekly Sync
- **trigger:** manual
- **when:** user asks "run a sync" or "what did we do this week"
- **facilitator:** Holden
- **participants:** all active agents
- **purpose:** Summarize recent decisions, open items, knowledge gaps

### Architecture Review
- **trigger:** manual
- **when:** user asks "architecture review" or introduces a new system/service
- **facilitator:** Holden
- **participants:** Naomi, Miller, Amos
- **purpose:** Review design decisions, flag concerns, align on patterns
